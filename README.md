# Prueba de ingreso

En este proyecto se utilizo:

 - Retrofit
 - MVVM
 - Dagger hilt
 - Navigation components
 - Coroutines
 - Marvel API (https://developer.marvel.com)

El proyecto esta compuesto por 3 modulos:
- Splash screen
- Lista de personajes
- Perfil de personaje
